## Sıradaki tek hamle (Şeyma — Cloud Shell, ~15 dk)

```bash
git clone https://github.com/seymaakrs/customer_agent && cd customer_agent
git checkout claude/eloquent-hamilton-1x0qfe   # veya merge sonrası main

echo -n '<RESET-SONRASI-NEON-STRING>' | gcloud secrets create lead-db-url --data-file=-
openssl rand -hex 32 | tee /tmp/ingest.txt | gcloud secrets create ingest-shared-secret --data-file=-

gcloud run deploy lead-ingestion \
  --source services/lead-ingestion --region europe-west1 --allow-unauthenticated \
  --set-env-vars MAIL_FROM=hello@slowdaysai.com,MAIL_TO=hello@slowdaysai.com \
  --set-secrets INGEST_SHARED_SECRET=ingest-shared-secret:latest,DATABASE_URL=lead-db-url:latest

URL=$(gcloud run services describe lead-ingestion --region europe-west1 --format 'value(status.url)')
curl $URL/healthz   # 200 beklenir
curl -X POST $URL/leads/ingest -H "Content-Type: application/json" \
  -H "X-Ingest-Secret: $(cat /tmp/ingest.txt)" \
  -d '{"source":"Site","external_id":"smoke-1","ad_soyad":"Test Kisi","sektor":"Otelcilik","konum":"Bodrum","butce":"15000"}'
# Aynı isteği tekrar at → "deduped": true beklenir (idempotency kanıtı)
```

Doğrulama: Neon SQL Editor → `SELECT * FROM leads;` → smoke-1 satırı.
